### ✨ Features
- 🔄 **Real-Time 24/7 Sync**: Automatically scrapes and updates every 4 hours via GitHub Actions.
- ⚡ **High Performance**: Optimized for fast loading with zero lag.
- 📺 **Massive Selection**: Merges top sources from across GitHub into one powerful list.
- 🧹 **Smart Cleanup**: Automatically removes duplicates and broken metadata.
- 📱 **Universal Support**: Works on all IPTV players, Smart TVs, and PCs.
