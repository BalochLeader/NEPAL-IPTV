### 🌟 Features
- 🔄 **24/7 Auto-Update**: Scrapes and merges playlists every 4 hours.
- 🚀 **High Performance**: Optimized M3U playlist for fast loading.
- 📺 **Massive Channel List**: Combines top sources like Mrgify-BDIX and PiratesTv.
- 🛠️ **Fully Automated**: Powered by GitHub Actions.

### 🛠️ How it Works
This bot uses a Python-based scraper to fetch the latest M3U links from multiple high-quality sources. It then merges them, removes duplicates, and updates the `nepal_iptv.m3u` file automatically.

**Sources Integrated:**
1. [Mrgify-BDIX-IPTV](https://github.com/abusaeeidx/Mrgify-BDIX-IPTV)
2. [PiratesTv](https://github.com/FunctionError/PiratesTv)
